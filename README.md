# H0HLAND PC CLUB

> **A modern single-page website for a gaming PC club**  
> *Built as a frontend development practice project*

---

## 📖 About This Project

This is a **practice project** created to strengthen my frontend development skills. The goal was to build a fully functional, visually appealing single-page website using only the core web technologies — **HTML, CSS, and vanilla JavaScript** — without relying on any frameworks or libraries.

The website represents **H0HLAND PC CLUB**, a fictional gaming lounge where players can book PCs, explore different zones, and get in touch with the club.
